### Development

Run this in your terminal:

```bash
git clone http://github.com/erming/jsonresume-theme-flat
cd jsonresume-theme-flat/
sudo npm install
npm start

# Open 'http://localhost:8080'
# You can now start editing 'resume.template'
```
